# Взаимодействие с объектами кроме рядов ячеек

**map_set_color** – установить цвет элемента.
```json
[{"tag": "arr1",
  "color": "#000000"},
 {"tag": "btn1",
 "color": "#000000"}]
```
**map_map_set_visibility** – включить/выключить видимость элемента. 
```json
[{"tag": "arr2",
  "visible": 1},
 {"tag": "btn2",
 "visible": 1}]
```
